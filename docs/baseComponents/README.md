# 安装&使用

::: tip 提示

@wocwin/t-antd-ui 是基于 vue2 + ant-design-vue 二次封装的基础组件

:::

## Install

```js
npm install @wocwin/t-antd-ui -S
&
yarn add @wocwin/t-antd-ui -S
```

## Usage
> ### 前提条件是：全局注册ant-design-vue组件库

```js
// main.js
import TAntdui from "@wocwin/t-antd-ui"

Vue.use(TAntdui)
```
